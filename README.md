# Little Lemon

## Overview

Welcome to the Little Lemon Restaurant website project! This site is designed to provide a delightful and engaging online presence for our fictional Little Lemon Restaurant. It features an elegant layout, enticing imagery, and information about the restaurant's menu, booking, and opening hours.

## Files

- `index.html`: The main HTML file for the restaurant website. It includes sections for promotions, the menu, booking, and opening hours.
- `styles.css`: The stylesheet that defines the visual appearance of the website. It includes styles for layout, color scheme, typography, and interactive elements.

## Features

### HTML

- **Semantic Structure**: Uses HTML5 semantic elements for a clear, accessible structure.
- **Responsive Navigation**: A navigation bar that links to the different sections of the website.
- **Engaging Content**: Sections for a promotional offer, menu highlights, booking information, and opening hours.

### CSS

- **Elegant Design**: A clean and modern design with a soothing color palette and typography.
- **Hover Effects**: Subtle hover effects on navigation links and cards for a more interactive user experience.
- **Card Zoom**: A gentle zoom effect on the `.card` elements when hovered, enhancing the visual engagement.

## Setup

1. **Clone the repository**: Clone or download the project to your local machine.
2. **Navigate to the project directory**: Change into the project directory after cloning.
3. **Open `index.html`**: Use any web browser to open the `index.html` file to view the project.

## Customization

Feel free to customize the HTML and CSS files to fit your preferences or the specific needs of your restaurant. You can change the text, images, colors, and more to personalize your website.
